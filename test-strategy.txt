| Test Type               | Description                                                                | Tool/Method                         |
| ----------------------- | -------------------------------------------------------------------------- | ----------------------------------- |

| **Integration Testing** | Ensure data flow between login and upload stages works correctly           | Token reuse and upload verification |
| **Regression Testing**  | Confirm no change breaks existing flows                                    | Script reruns under GitHub Actions  |
| **Negative Testing**    | Simulate invalid credentials or versions                                   | Controlled variable alteration      |
